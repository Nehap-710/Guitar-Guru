Music theory visualization

Learn music from scratch with easy chord and scale visualizations and fretboard positions for each chord.

Chords

The fretboard below shows the location of every note that occurs in a given chord along the length of the fretboard. In the dropdown menus underneath the fretboard, you can select which chord's notes you want to display. Here you can specify (i) the root note, (ii) the type of chord, and (iii) the tuning that the guitar is in.

Scales

The fretboard below shows the location of every note that occurs in a given scale along the length of the fretboard. In the dropdown menus underneath the fretboard, you can select the key for which you want to display a scale. Here you can specify (i) the root note, (ii) the mode of the key, and (iii) the tuning that the guitar is in.
